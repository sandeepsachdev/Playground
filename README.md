# Prompts

The prompts used to create this app, in order:

1. > Create a spring boot application which provides a tv guide for free to air
   > Sydney tv. It should Also provide some highlights on what to watch each day
   > between 6pm and 10pm each night.

2. > Add a dockerfile suitable for deployment to render

3. > Can you plugin a real.source of live data

4. > Remove the static schedule so it's no data shown when no lounge feed is
   > available. Shows dates for the guide.

5. > delete all source code from this project and start a new project from
   > scratch.

6. > just delete the java/templates. The new project will also be spring boot

7. > delete all branches and start using main for all comments. The new project
   > should use various news feeds that are available. It should filter out
   > common words like a, this, and etc and display a word cloud based on
   > trending phrases and words.
